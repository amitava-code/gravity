# Drop The Letters

Interactive physics-based typography. Built with Matter.js (gravity/collisions)
and opentype.js (turns font glyphs into physics bodies).

- Click once to release the title and let it fall
- Click again to drop random letters
- Double-click to rain the whole word from the sky
- Drag any letter around

## Run locally
Just open `index.html` in a browser — no build step needed.
